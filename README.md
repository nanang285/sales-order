## 📜 Description

Zen Landing Pages is a project built with Laravel 11 that includes a robust **password reset** feature using Gmail SMTP. This project demonstrates how to configure Laravel to send emails via Gmail without registering an application key with Google.

## 🎨 Features

- 🔒 **Secure Password Reset**: Users can reset their passwords via email securely.
- 📱 **Responsive Design**: Fully responsive and mobile-friendly.
- 🛠️ **Built with Laravel 11**: Leveraging the latest Laravel features.
